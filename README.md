# Algorithm-Development-in-C
Practical C programming exercises focused on algorithmic logic, problem-solving, and efficient code implementation.

Note: The source code and exercises are documented in Portuguese as they were developed during my technical courses.

Here you'll find topics covering:

Functions, Recursion, Matrices & Arrays, Pointers, Data Types in C, etc...

This project was developed and tested using the following setup:

* **IDE:** Code::Blocks (Version 25.03)
* **Compiler:** GCC (MinGW-w64)
* **OS:** Windows 11
* **Language Standard:** C11 (ISO/IEC 9899:2011)

### How to Compile and Run
1. Ensure you have **Code::Blocks** and a **GCC compiler** installed.
2. Clone the repository.
3. Open the `laboratorio.workspace` file in Code::Blocks.
4. Select the project you wish to run, set it as "Active Project", and press **F9**.
